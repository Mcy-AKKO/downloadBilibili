# B站下载器

一个简单易用的 B 站视频下载工具，网页界面操作，粘贴视频链接或 BV 号即可下载。基于 Flask + yt-dlp 实现。

## 功能特点

- 🔍 **粘贴即用**：支持视频链接、`b23.tv` 短链或直接输入 BV 号，自动识别
- 📋 **下载前预览**：显示视频标题、UP 主、时长、封面和可选清晰度
- 🎬 **高清下载**：最高 1080p，优先 H.264 编码（兼容性好），自动合并音视频为 mp4
- 📊 **实时进度**：下载进度、速度、剩余时间实时显示
- 📁 **自定义保存**：可选择保存目录，支持为每个视频指定文件夹名
- 📃 **下载管理**：已下载视频列表，一键在资源管理器中打开
- 📦 **开箱即用**：提供打包好的 exe，无需安装 Python 环境

## 下载使用（推荐）

前往 [Releases 页面](https://github.com/Mcy-AKKO/downloadBilibili/releases) 下载最新的 `BilibiliDownloader.exe`，双击运行即可——程序会自动在浏览器中打开操作界面（`http://127.0.0.1:5000`）。

> 首次运行时 Windows 可能弹出防火墙提示，允许即可（程序只在本机 127.0.0.1 监听，不对外开放）。

## 从源码运行

需要 Python 3.8+：

```bash
git clone https://github.com/Mcy-AKKO/downloadBilibili.git
cd downloadBilibili
pip install -r requirements.txt
python main.py
```

运行后浏览器会自动打开 `http://127.0.0.1:5000`。

## 使用方法

1. 在输入框粘贴 B 站视频链接（或 BV 号），点击解析
2. 确认视频信息无误后，点击下载（可选：填写自定义文件夹名）
3. 等待进度条完成，即可在下载列表中找到视频并打开所在文件夹

视频默认保存在程序目录下的 `downloads` 文件夹，可在设置中更换保存路径。

## 常见问题

- **只能下载免费公开视频。** 本工具不支持、也请勿尝试下载大会员专享、付费课程等内容。
- **部分视频清晰度受限**：未登录状态下 B 站对可用清晰度有限制，属正常现象。
- **下载失败**：请确认视频存在且为公开视频，或稍后重试。
